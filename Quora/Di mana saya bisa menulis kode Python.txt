Ada banyak cara yang bisa digunakan untuk menulis Python script, salah satu tools yang populer yaitu menggunakan Jupyter notebook. Jupyter notebook akan terinstall otomatis apabila kita menginstall Python menggunakan Anaconda. Jika kita menginstall Python tidak melalui Anaconda, kita bisa menginstall Jupyter notebook terlebih dahulu menggunakan pip atau conda. (Script : pip install jupyter)
Salah satu keunggulan Jupyter notebook adalah kita bisa melihat integrasi antara input dan output dari script yang kita tulis, di dalam satu tampilan dokumen yang interaktif
